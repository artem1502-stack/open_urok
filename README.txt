Хеллоу. Данный проект писался на бегу и проблемы под сборку во что-то простое под Windows имеются
Если я не скинул иное - вот наш план действий

1. Установить python не моложе 3.9 с сайта:
https://www.python.org/downloads/
2. Запустить в консоли следующую команду:

python -m pip install -r requirements.txt

3. Далее - запуск самого скрипта:

Необходимо перейти в саму папку open_urok в консоли
Сделать это можно командой:

cd ПУТЬ_ДО_ПАПКИ

Далее:

python main.py

Вуаля. Если не прислал другой версии - сори, машина под винду сдохла и жить не планирует, а нормально без неё
собрать экзешник - проблематично
